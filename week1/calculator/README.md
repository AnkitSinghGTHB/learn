Build a simple command-line calculator!

## Requirements
- Add, subtract, multiply, and divide two numbers
- Ask user for the operation and numbers
- Display the result
- Handle division by zero

## Getting Started

1. Copy `calculator_starter.py` to `calculator.py`
2. Complete the TODO sections
3. Run with `python calculator.py`

## Example Output
```
=== Simple Calculator ===
Enter first number: 10
Enter second number: 5
Choose operation (+, -, *, /): *
Result: 10 * 5 = 50
```
